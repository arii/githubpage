How to embed a video to your github page

Unfortunately, it is not possible to embed a youtube video with markdown, however you can embed a video by directly editing the html. This tutorial will show you how to modify the html to embed your video.

Steps to add the video

  1. First create a github page for your repository. Go to your repository's settings and select "Launch automatic page generator". Follow the instructions to create your page and familiarize yourself with markdown.
  2. Create your website with markdown and include a section for your video. Publish the page.
  3. Upload your video to youtube. Select the share option and choose embed video. It should look like this:
<iframe width="560" height="315" src="https://www.youtube.com/embed/f5l8GA1PHm8" frameborder="0" allowfullscreen></iframe>
  1. Check out the github pages branch of your repository (gh-pages). Here's the code I used to check out this repository, "githubpage":
git clone -b gh-pages https://github.com/arii/githubpage.git
  1. Edit index.html with your favorite text editor. Locate the video section and paste the iframe code from youtube.
  2. Commit and push your changes.
git add index.html
git commit -m "added video"
git push
